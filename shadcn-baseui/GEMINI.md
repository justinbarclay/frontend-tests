# Project Instructions: shadcn-baseui

This project is built using **shadcn**, **Base UI**, and **Tailwind CSS**.

## Tech Stack

- **UI Components**: shadcn (using Base UI primitives)
- **Styling**: Tailwind CSS
- **Framework**: React

## Workflows

- **Component Management**: Use the `shadcn` CLI for adding and managing components.
- **MCP Tools**: Use the configured `shadcn` MCP server to search and view available components and registries.
